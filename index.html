Prework Study Guide

HTML

  • The head element contains information about the webpage (ie metadata).
  • The section element seperates different topics within the webpage.
  • li- list item; ul - unordered list; ol - ordered list.

CSS

  • You can use CSS three ways; Inline, Internal Sheet, and External Sheet.
  • Using * as a selector makes ALL elements on the page have the defined attributes.

GIT

  • Check branch with git-status command
  • change branch with checkout command
  • -b flag creates a new branch
  • feature/(insert name of new branch) add feature prefix to indicate that this a feature.
  • Ex: git checkout -b feature/(insert name of branch).
  • git pull origin main: pulls any changes that you don't have locally from the remote base branch.
  • git push origin (whatever you're trying to push): pushes changes to your remote branch.
  • git checkout -b branch-name-here: creates a new branch and switches to it.
